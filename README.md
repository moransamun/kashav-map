# מפת ניהול המשבר בקש״ב — גרסה מהודקת (אב-טיפוס)

## למי שרוצה להתנסות בכלי
פתחו את `index.html` בכל דפדפן (גם בלי אינטרנט, אחרי ההורדה) — זו עמוד
אחד עצמאי. שלושה שלבים בבחירות בלבד (בלי הקלדה), ובסוף מפה אישית עם
המלצות. שום מידע לא נשלח לשום מקום בהתנסות הזו — הכפתור "שליחת המפה
למשרד" בסוף הוא הדגמה בלבד ולא שולח בפועל.

כדי להתנסות דרך הדפדפן בלי להוריד כלום, אחרי שהקובץ יעלה לריפו ויופעל
GitHub Pages (הוראות למטה), הכלי יהיה זמין בקישור ציבורי.

## מה יש כרגע בריפו — ולמה ההעלאה לא "נתפסה"
בדקתי: הריפו `moransamun/kashav-map` מכיל כרגע **רק קובץ `README.md`
ריק**. אין בו `index.html` בכלל — כך שאין קובץ קיים "להחליף"; צריך
**להוסיף** קובץ חדש, לא להעלות על גבי קובץ שלא קיים.

## שלב 1 — להעלות את index.html לריפו
1. נכנסים ל-`github.com/moransamun/kashav-map`.
2. **Add file → Upload files**.
3. גוררים את `index.html` מהזיפ (חשוב: השם חייב להיות בדיוק `index.html`,
   עם i קטנה, באנגלית — לא "Index" ולא עם רווחים).
4. גוררים למקום ה-root של הריפו (לא לתוך תיקייה).
5. **Commit changes**.

## שלב 2 — להפעיל GitHub Pages (אם עדיין לא מופעל)
זו הסיבה השנייה שהדף לא "עולה" גם אם הקובץ קיים:
1. בריפו: **Settings → Pages**.
2. תחת "Build and deployment" → Source: **Deploy from a branch**.
3. Branch: **main**, תיקייה: **/ (root)**. שומרים.
4. אחרי כ-1-2 דקות הדף יהיה זמין ב-
   `https://moransamun.github.io/kashav-map/`.

## אם עדיין לא עובד
בדקו ב-**Actions** tab בריפו אם יש שגיאת build אדומה, ותעדכנו אותי מה
היא אומרת — אני יכולה לעזור לפתור משם.
