# מפת ניהול המשבר בקש״ב — גרסה מהודקת (אב-טיפוס)

קובץ `kashav-map.html` הוא גרסה מקוצרת ואינטראקטיבית של הכלי הקיים בריפו:
3 שלבים במקום 5, מבוסס בחירות (צ׳יפים) במקום כתיבה חופשית, עם רמזור
אוטומטי, המלצות קש״ב שנוצרות לפי הבחירות, וניתוב עדין למי שלא מתאים
למשרד. הקובץ עצמאי לגמרי — HTML אחד, בלי תלות בשרת או בקבצים חיצוניים.

## מה זה לא
זה **אב-טיפוס להשוואה** מול הכלי הקיים, לא תחליף סופי. שדה איסוף
הפרטים בסוף (שם/טלפון/ערוץ) הוא הדגמה בלבד ואינו שולח בפועל ל-WhatsApp
או למייל — יש לחבר אותו ליעד השליחה האמיתי לפני עלייה לאוויר.

## איך להעלות ל-GitHub לפרויקט הקיים

אין לי גישת כתיבה לריפו שלך (`moransamun/kashav-map`) — רק קריאה. כדי
להעלות בעצמך:

1. הורידי את `kashav-map.html` (ואת `README.md` הזה, אם רוצים) מהצ׳אט.
2. בריפו ב-GitHub, אפשר בלי טרמינל: **Add file → Upload files**, גררי
   את `kashav-map.html` לתיקייה הרצויה (למשל כ-`v2/index.html` או
   כקובץ נפרד לצד `index.html` הקיים כדי להשוות), ו-**Commit changes**.
3. אם יש לך את הריפו משוכפל (cloned) מקומית:
   ```bash
   git add kashav-map.html README.md
   git commit -m "גרסה מהודקת של מפת קש״ב"
   git push
   ```
4. אם הריפו מוגש דרך GitHub Pages, הדף יהיה זמין תוך דקה-שתיים בכתובת
   המתאימה (למשל `https://moransamun.github.io/kashav-map/kashav-map.html`).
